## 🧪 What RxnCaption Does

RxnCaption is designed to help with reaction diagram parsing through visual prompt guided captioning. In plain terms, it looks at a reaction image and produces text that describes what it sees.

Typical use cases:

- Reading reaction schemes from papers
- Checking diagram content
- Turning image-based chemistry notes into text
- Helping with study and review work
- Reviewing reaction diagrams in a faster way

## ⚙️ Basic Use

1. Open the app.
2. Choose an image file such as `.png`, `.jpg`, or `.jpeg`.
3. Wait while the diagram is analyzed.
4. Review the caption text.
5. Copy, save, or reuse the result.

For better output, use clear images with sharp text and good contrast. Cropped diagrams usually work better than full page scans.

## 🧩 Tips for Better Results

- Use a clean image with no blur.
- Keep the whole reaction diagram in view.
- Avoid dark shadows and glare.
- Use high-resolution files when you can.
- Crop out pages, margins, and unrelated figures.
- If the image is sideways, rotate it before loading it.

## 🔄 Supported Input

RxnCaption is meant for reaction diagram images. It may work best with:

- Reaction schemes from journal papers
- Scanned lab notes
- Diagram screenshots
- Cropped chemistry figures
- Slides with reaction pathways
